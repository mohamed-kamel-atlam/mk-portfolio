## Overview

A modern, responsive, and interactive Tic-Tac-Toe game built with vanilla JavaScript and Tailwind CSS.

## Features

- **Interactive Gameplay** - Play classic Tic-Tac-Toe with instant feedback and smooth animations
- **Score Tracking** - Real-time scorekeeping for both players (X and O)
- **Beautiful Animations** - Smooth animations with decorative X and O icons that animate throughout gameplay
- **Fully Responsive** - Plays perfectly on all devices - mobile, tablet, and desktop
- **Draw Detection** - Automatically detects draw situations and alerts players
- **Reset Game** - Quick reset button to start new games while keeping score history

## How to Play

- The game is played on a 3x3 grid
- Player X always goes first
- Players alternate placing their mark (X or O) in empty squares
- The first player to get three in a row (horizontal, vertical, or diagonal) wins
- If all squares are filled without a winner, the game is a draw
- Click the "Reset" button to start a new game

## Responsive Design

This project features a mobile-first responsive design that adapts seamlessly to all screen sizes:

- **Mobile (< 640px)** - Full-width layout optimized for touch
- **Tablet (640px - 1024px)** - Adjusted spacing and font sizes
- **Desktop (> 1024px)** - Full viewport optimization with premium spacing
