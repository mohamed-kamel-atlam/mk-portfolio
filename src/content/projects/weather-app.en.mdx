## Overview

A modern weather application built with React and Vite, supporting Arabic and English with real-time data from the OpenWeatherMap API.

> This is the first project I built after completing the React course in November 2025 — a simple but complete app that puts the fundamentals into practice.

## Key Features

- Real-time temperature (current, min, max)
- Arabic and English language support via i18next
- Localized date and time formatting with moment.js
- Redux Toolkit for state management
- Tailwind CSS v4 for styling
- Fully responsive design

## Architecture

The app is built on React 19 and Vite 7, with Redux Toolkit handling application state and an API slice responsible for fetching current conditions from the [OpenWeatherMap Current Weather API](https://openweathermap.org/current). Internationalization is handled through i18next, driving both the Arabic/English translations and localized date formatting, while Tailwind CSS v4 covers the responsive styling layer.
