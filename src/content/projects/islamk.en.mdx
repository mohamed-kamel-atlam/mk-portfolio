## Overview

إسلامك (Islamk) هو تطبيق إسلامي عصري لأوقات الصلاة والذكر اليومي.

Islamk is a modern Islamic web app for prayer times and daily dhikr, built with React 19, Vite 7, and Tailwind CSS v4.

## Features

- **Prayer Times** — أوقات الصلاة لأكثر من 10 مدن عربية عبر Aladhan API. Prayer times for 10+ Arab cities via the Aladhan API.
- **Dhikr Counter** — عداد أذكار يومي مع حفظ البيانات في localStorage. A daily dhikr counter that persists progress in `localStorage`.
- **Dark Mode** — وضع ليلي مع حفظ تفضيل المستخدم. A dark mode toggle with the user's preference remembered across sessions.
- **Arabic Date** — التاريخ الميلادي بالعربية عبر moment.js. Gregorian dates formatted in Arabic using moment.js.
- **Responsive Layout** — متوافق مع جميع أحجام الشاشات. Layouts adapt across all screen sizes.

### Supported Cities

القاهرة، الرياض، أبو ظبي، الرباط، الخرطوم، الكويت، الجزائر، بيروت، بغداد، صنعاء

## API Integration

The app relies on the [Aladhan Prayer Times API](https://aladhan.com/prayer-times-api) for timing data:

```http
GET https://api.aladhan.com/v1/timingsByCity/{date}?city={city}&country={country}&method=4
```

| Parameter | Description                                  |
| :-------: | :------------------------------------------- |
|  `date`   | Date in `YYYY-M-D` format                    |
|  `city`   | City name in English                         |
| `country` | Country name in English                      |
| `method`  | `4` = Umm al-Qura calculation method (Mecca) |

## RTL / Arabic UI Approach

The interface is built around a right-to-left, Arabic-first reading experience, pairing Arabic copy with the Amiri typeface (bundled in regular, bold, italic, and bold-italic weights) to keep prayer times and dhikr text legible and true to Arabic typographic conventions. Dates and other locale-sensitive content are formatted in Arabic via moment.js rather than defaulting to a Latin/Gregorian-only presentation.

## Design & UX

Styling is built on Tailwind CSS v4, with a dark mode that respects the user's saved preference and a responsive layout that scales from mobile to desktop. Navigation between the home, prayer times, and dhikr counter views is handled with React Router.

## Highlights

- React 19 + Vite 7 for a fast, modern SPA foundation.
- Tailwind CSS v4 for styling, with Axios for HTTP requests and React Icons for iconography.
- Persisted dhikr counts and theme preference via `localStorage`, avoiding the need for a backend for simple state.
- Arabic-first RTL design paired with a dedicated Arabic typeface (Amiri) rather than a generic system font.
