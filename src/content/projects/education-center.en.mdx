## Overview

The Education Center Management System (ECMS) is an operational admin platform
for modern education centers — a single portal to sign in, register a center,
and run day-to-day operations. It centers on secure authentication, role-based
dashboards, administration, and analytics.

## Problem

Education centers juggle enrollment, staff, and reporting across disconnected
tools. They needed one operational backbone that streamlines administration and
surfaces the metrics that matter — without a steep learning curve.

## Solution

A focused React + TypeScript frontend: a polished authentication flow (sign in /
register a new center), a role-aware administrative dashboard, and analytics
views — built on a reusable component system and a clean, token-driven UI.

## My Role

As Front-End Engineer I built the authentication and onboarding flows, the
administrative dashboard and its role-based views, and integrated the REST APIs,
backing it all with a component library that kept the portal consistent.

## Challenges

Keeping the interface approachable while covering administrative depth: the
dashboard had to stay legible for non-technical staff yet expose the controls
administrators actually need.

## Lessons Learned

A clear component system and consistent design tokens made the admin surface
fast to extend — new sections slotted in without a redesign, which is exactly
what an operational tool needs as it grows.
