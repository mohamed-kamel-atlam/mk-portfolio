## Overview

A web application that displays daily Islamic prayer times based on the user's selected city. It was built to practice and apply skills from an Integrate APIs course, focusing on consuming a third-party REST API and rendering the results dynamically.

## Features

- City-based prayer time lookup
- Displays all six daily prayers: Fajr, Sunrise, Dhuhr, Asr, Maghrib, and Isha
- Shows the current date and time in Arabic
- Clean, responsive UI with a mosque background
- Right-to-left (RTL) Arabic interface

## API Integration

Prayer times are fetched from the [Aladhan Prayer Times API](https://aladhan.com/prayer-times-api), a free REST API for Islamic prayer times worldwide. The project uses Axios to make HTTP requests, parses the JSON responses, and dynamically renders the data to the DOM. It also integrates third-party UI libraries such as Bootstrap and Font Awesome alongside the API calls.

## Design & UX

The interface is built as a right-to-left (RTL) Arabic experience, with a mosque background image and a clean, responsive layout powered by Bootstrap components and Font Awesome icons.

## Highlights

- Practical application of HTTP requests with Axios
- Handling API responses and parsing JSON data
- Dynamically rendering fetched data to the DOM
- Building an RTL Arabic interface with third-party UI libraries
